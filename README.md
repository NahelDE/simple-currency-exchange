# Simple Currency Exchange

## Goal

Make a simple web app that shows the conversion of an user input into a chosen currency.
Example : the user input 10 euros and he want to know how much it is in US dollar, the application 
will calculate using API's number.

## Technologies Used

- Python 3.14
- Flask
- Jinja2
- https://github.com/fawazahmed0/exchange-api (the api i used)

## Software 

- PyCharm
